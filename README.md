# Alon to Intu

Created this project using create-react-app

On Github pages  
https://alongalperin.github.io/alon_intu/

How to run?

1. install dependencies
   ```
   npm install
   ```
2. run project
   ```
   npm start
   ```

How to run with backend-mock?  
Mock using MirageJS
```
npm run start-mirage
```

How to run tests?

```
npm run test
```

Things to improve?
